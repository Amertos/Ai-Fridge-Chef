## ğŸ“– About The Project

**AI Fridge Chef** is an intelligent web application designed to reduce food waste and solve the eternal question: *"What should I cook today?"*. 

By leveraging the power of **Google's Gemini AI**, this app allows users to simply snap a photo of their fridge content or enter ingredients manually. The AI then identifies the ingredients and suggests delicious, personalized recipes complete with step-by-step cooking instructions.

> "Your personal sous-chef that lives in your pocket."

## ğŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Google Cloud Project with Gemini API access

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ai-fridge-chef.git
   cd ai-fridge-chef
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```

5. **Open in Browser**
   Navigate to `http://localhost:5173` to start cooking!
